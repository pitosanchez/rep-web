# REP - Rare Renal Equity Project

## What This Is
A public accountability platform mapping how APOL1-mediated kidney disease and FSGS are shaped by place — poverty, food environment, care access, and structural inequity.

## Tech Stack
- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS
- MapLibre GL JS

## Key Principles
- No individual-level disease data
- Stories tied to geography, not identity
- Aggregation-first ethics
- Threshold before displaying stories (minimum 5)

## File Structure
- app/ — pages and routes
- components/ — React components
- lib/ — types and utilities
- public/data/ — GeoJSON files
